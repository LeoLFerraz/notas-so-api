Bem vindo à API de Notas de Sistemas Operacionais 2018.2 para BSI-UNIRIO

Comando Descrição Exemplo
Cadastrar CPF Cadastra um novo CPF na aplicação. Espera um JSON contendo o CPF via POST e retorna um JSON com o CPF cadastrado em caso de sucesso. POST na URL /api/cpfs/
{
"cpf": "12345678900"
}
Cadastrar Nota Cadastra uma nova nota atrelada a um dado CPF. Espera um JSON contendo a nota e o CPF via post e retorna um JSON com o CPF cadastrado em caso de sucesso. POST na URL /api/cpfs/{cpf}/notas/
{
"nota": 10.0
}
Consultar Notas Espera uma requisição padrão GET. Retorna um JSON com as notas do CPF consultado. GET na URL /api/cpfs/{cpf}/notas/
Consultar Média Espera uma requisição padrão GET. Retorna um JSON com a média aritmética das notas do CPF consultado. GET na URL /api/cpfs/{cpf}/media/}
